[접근지정자] [제한자] class 클래스명
접근지정자 생략가능한데, 안쓴 경우에는 디폴트로 가정


접근지정자는 네가지 중에 하나 
제한자는 두가지 (일단 ,초급자바에서는 두가지만 @) abstract / final
	abstract 추상클래스 만들어줌 – 내부에 추상메소드 하나라도 가지고 있으면 abstract (몸통이 없는 메소드 (중괄호가 없는)_ 내용이 없고 제목만 있는 메소드)
		이 클래스는 절대로 객체로 생성 될 수 없음 (불완전한 클래스이기 때문에)
		몸통이 없어서 몸통을 매달아야 하는데, 자식클래스에서 상속되어서 오버라이드 해야함
	final 메소드, 클래스, 변수 앞에 붙을 수 있음
		1. 변수 앞에 붙으면 상수가 됨 -> = 연산자 왼편에 올 수 없음 (=연산자 왼편에는 변수만 올 수 있음 @@!!!)
			ex) double final PI = 3.141592; (3.141592라는 값을 저장할 때 딱 한번만 왼편에 올수 있음)
				따라서 PI= PI*2는 될수 없음!!!
		2. 클래스 앞에 붙으면 상속 금지 -> 자식 클래스를 가질 수 없음
			(종단클래스 됨 )
